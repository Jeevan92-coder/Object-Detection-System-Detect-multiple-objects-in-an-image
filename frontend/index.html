COCO Dataset — Setup Guide

STEP 01
Download COCO 2017
python scripts/download_coco.py --split val --samples 100
Downloads annotations + 100 validation images (~50 MB) for quick testing
STEP 02
Folder Structure
object-detection-system/
└── data/
    └── coco/                ← place here
        ├── images/
        │   ├── train2017/
        │   └── val2017/
        └── annotations/
            ├── instances_train2017.json
            └── instances_val2017.json
STEP 03
Run Evaluation
python scripts/evaluate_coco.py --split val
Computes mAP@0.5, mAP@0.5:0.95 and per-class AP on COCO validation set
