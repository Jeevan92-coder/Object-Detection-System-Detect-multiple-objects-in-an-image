GET /api/health

Service health check. Returns model info, device status, and total class count. Use this to verify your backend is running before making requests.

Response 200 OK
{ "status":"healthy", "model":"yolov8n.pt", "device":"cpu", "classes":80 }

GET /api/info

Returns detailed model configuration, list of supported input formats, and the full array of 80 COCO class names sorted by class ID.

Response 200 OK
{ "model":"yolov8n.pt", "classes":["person","bicycle","car",...], "formats":["jpg","png","bmp","webp"] }

POST /api/detect/image

Detect objects in a single uploaded image. Returns annotated result image URL and full detection data with bounding boxes, confidence scores, and class summaries.

Form Fields
image       File    required  JPG / PNG / BMP / WEBP
confidence  float   optional  default 0.25
iou         float   optional  default 0.45
classes     string  optional  comma-separated class filter
cURL Example
curl -X POST http://localhost:5000/api/detect/image \
  -F "image=@photo.jpg" \
  -F "confidence=0.3" \
  -F "classes=person,car"
Response 200 OK
{
  "success": true,
  "data": {
    "total_objects": 3,
    "detections": [{
      "class_id": 0,
      "class_name": "person",
      "confidence": 0.91,
      "bbox": { "x1":100, "y1":50, "x2":250, "y2":400 },
      "area_px": 52500,
      "center": { "x":175, "y":225 }
    }],
    "class_summary": { "person":2, "car":1 },
    "result_image_url": "/api/results/result_abc.jpg",
    "inference_time_ms": 4.7
  }
}

POST /api/detect/frame

⭐ New in v2 — For live camera: send a base64-encoded JPEG frame, get back detections + annotated frame as base64. Designed for ultra-low-latency webcam integration.

JSON Body
{
  "frame":      "data:image/jpeg;base64,/9j/4AAQ...",
  "confidence": 0.25,
  "iou":        0.45,
  "classes":    ["person","car"]  // optional filter
}
Response 200 OK
{
  "success": true,
  "data": {
    "total_objects": 2,
    "detections":    [...],
    "class_summary": { "person":1, "car":1 },
    "frame_b64":     "/9j/4AAQ...",  // annotated JPEG
    "image_size":    { "width":1280, "height":720 }
  }
}
JavaScript (Webcam Integration)
// Capture frame from video element
offCanvas.drawImage(video, 0, 0, w, h);
const b64 = offCanvas.toDataURL("image/jpeg", 0.6);

const res = await fetch("/api/detect/frame", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ frame: b64, confidence: 0.25 })
});
const { data } = await res.json();

// Draw annotated frame back onto visible canvas
const img = new Image();
img.src = "data:image/jpeg;base64," + data.frame_b64;
img.onload = () => ctx.drawImage(img, 0, 0);

POST /api/detect/batch

Process up to 10 images in a single request. Returns per-image detection results plus aggregate statistics including total objects, class distribution, and total inference time.

cURL Example
curl -X POST http://localhost:5000/api/detect/batch \
  -F "images=@img1.jpg" \
  -F "images=@img2.jpg" \
  -F "images=@img3.jpg" \
  -F "confidence=0.3"
Response 200 OK
{
  "success": true,
  "data": {
    "total_images": 3,
    "results": [
      { "original_filename":"img1.jpg", "total_objects":4, "inference_time_ms":5.1, ... },
      { "original_filename":"img2.jpg", "total_objects":2, "inference_time_ms":4.8, ... }
    ]
  }
}

GET /api/classes?search=car

List all 80 COCO class names, optionally filtered by a search query. Returns class IDs and names sorted alphabetically.

Response 200 OK
{ "classes": [
    { "id": 2, "name": "car" },
    { "id": 7, "name": "truck" }
  ], "count": 2
}

🐍 Python Example

Python — requests library
import requests

# Single image detection
with open("photo.jpg", "rb") as f:
    response = requests.post(
        "http://localhost:5000/api/detect/image",
        files={"image": f},
        data={"confidence": 0.3, "classes": "person,car"}
    )

data = response.json()["data"]
print(f"Found {data['total_objects']} objects")

for det in data["detections"]:
    print(f"  {det['class_name']}: {det['confidence']:.1%}")

⚡ JavaScript Example

JavaScript — Fetch API
const fd = new FormData();
fd.append("image", fileInput.files[0]);
fd.append("confidence", "0.3");
fd.append("classes", "person,car,dog");

const response = await fetch("/api/detect/image", {
  method: "POST",
  body: fd
});

const { data } = await response.json();
console.log(`Detected ${data.total_objects} objects`);

data.detections.forEach(d => {
  console.log(`${d.class_name}: ${(d.confidence * 100).toFixed(1)}%`);
});
