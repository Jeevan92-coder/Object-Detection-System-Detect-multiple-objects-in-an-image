GET /api/health

Service health check. Returns model info and device status.

Response
{ "status":"healthy", "model":"yolov8n.pt", "device":"cpu", "classes":80 }

GET /api/info

Returns model configuration, supported input formats, and the full list of class names.

POST /api/detect/image

Detect objects in a single uploaded image. Returns annotated result image URL and detection data.

Form Fields
image       File    required  JPG/PNG/BMP/WEBP
confidence  float   optional  default 0.25
iou         float   optional  default 0.45
classes     string  optional  comma-separated class filter
cURL
curl -X POST http://localhost:5000/api/detect/image \
  -F "image=@photo.jpg" -F "confidence=0.3"
Response
{ "success": true, "data": {
  "total_objects": 3,
  "detections": [{
    "class_id": 0, "class_name": "person",
    "confidence": 0.91,
    "bbox": {"x1":100,"y1":50,"x2":250,"y2":400},
    "area_px": 52500, "center": {"x":175,"y":225}
  }],
  "class_summary": {"person":2,"car":1},
  "result_image_url": "/api/results/result_abc.jpg",
  "inference_time_ms": 4.7
}}

POST /api/detect/frame

⭐ New in v2 — For live camera: send a base64-encoded JPEG frame, get back detections + annotated frame as base64. Designed for low-latency webcam integration.

JSON Body
{
  "frame":      "data:image/jpeg;base64,/9j/4AAQ...",
  "confidence": 0.25,
  "iou":        0.45,
  "classes":    ["person","car"]  // optional filter
}
Response
{ "success": true, "data": {
  "total_objects": 2,
  "detections":    [...],
  "class_summary": {"person":1,"car":1},
  "frame_b64":     "/9j/4AAQ...",  // annotated JPEG
  "image_size":    {"width":1280,"height":720}
}}
JavaScript (browser webcam)
// Capture frame from video element
offCanvas.drawImage(video, 0, 0, w, h);
const b64 = offCanvas.toDataURL("image/jpeg", 0.6);

const res = await fetch("/api/detect/frame", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ frame: b64, confidence: 0.25 })
});
const { data } = await res.json();

// Draw annotated frame back onto visible canvas
const img = new Image();
img.src = "data:image/jpeg;base64," + data.frame_b64;
img.onload = () => ctx.drawImage(img, 0, 0);

POST /api/detect/batch

Process up to 10 images in a single request. Returns per-image results plus aggregate stats.

cURL
curl -X POST http://localhost:5000/api/detect/batch \
  -F "images=@img1.jpg" \
  -F "images=@img2.jpg" \
  -F "confidence=0.3"

GET /api/classes?search=car

List all COCO 80 classes, optionally filtered by a search query.

Python Example

Python
import requests

with open("photo.jpg", "rb") as f:
    r = requests.post(
        "http://localhost:5000/api/detect/image",
        files={"image": f},
        data={"confidence": 0.3}
    )

for det in r.json()["data"]["detections"]:
    print(f"{det['class_name']}: {det['confidence']:.1%}")

JavaScript Example
